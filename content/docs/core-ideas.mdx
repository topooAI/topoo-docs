Topoo is the core-ideas board for the whole documentation system.

It should answer the questions that sit above any one work surface:

- what Topoo is optimizing for
- why the system is split into separate boards
- how the boards relate to each other
- what belongs to shared system thinking versus board-specific implementation

The job of this page is not to describe a concrete feature.

Its job is to hold the stable framing that later solution boards build on top of.
